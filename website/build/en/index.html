原生

只需要引入一个 CSS 文件，即可使用大部分的功能，甚至并不需要任何的构建工具

原子化

只需要专注在 HTML 就可以开发大部分网页样式

简单

我们没有发明庞大且让人迷惑的 class 名字，只有屈指可数的命名规则

无依赖

不依赖任何框架，唯一的依赖就是这套命名规则

渐进增强

我们提供了 SASS，LESS mixin 帮助你更快的创建自定义代码（可选的）

更少，更快

这套命名规则让你写的代码就已经是压缩状态，意味着你可以写更少的代码做更多的事
