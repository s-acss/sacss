# Auxiliary methods

SACSS does not cover all the styles, the following four items commonly used styles, SACSS provides the corresponding Mixin, convenient for users to use.

1. `Clearfix`: Clear float. 2;
2. `Ellipsis`: single line of text beyond the display of `... `.
3. `Ellipsis-multiple`: multi-line text out of display `... `;
4. `Cell`: fill the remaining space in the floating layout;

![img](/img/demo.png)

``` less
@import 'sacss/less/helpers.less';

.clearfix{ #Clearfix;  }
.ellipsis{ #Ellipsis;  }
.cell{ #Cell; }
.ellipsis-mul{ #Ellipsis-multiple;  }
.ellipsis-mul._2row{ -webkit-line-clamp: 2; }
```

``` html
<div class="fs12 lh16 ellipsis">
    Life was like a box of chocolates. You never know what you're going to get.
</div>
<div class="fs12 lh16 ellipsis-mul _2row">
    Life was like a box of chocolates. You never know what you're going to get.
</div>
<div class="clearfix">
    <div class="fl mr16">float-left</div>
    <div class="cell">Life was like a box of chocolates. You never know what you're going to get.</div>
<div>
```
